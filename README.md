Falling Sand Simulation Written in C using Raylib

Tiles:
  Sand,
  Wet_Sand,
  Dirt,
  Water,
  Lava,
  Water_Vapor,
  Smoke,
  Glass,
  Volcanic Glass,
  Basalt,
  Stone(immovable),
  Wood(immovable)

Interaction:
  Sand + Water -> Wet_Sand,
  Dirt + Water -> Wet_Dirt,
  Water_Vapor -> Water,
  Lava + Sand -> Glass,
  Lava + Water -> Volcanic_Glass and Water_Vapor,
  Lava + Dirt -> Basalt
